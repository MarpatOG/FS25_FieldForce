# Windows App

Avalonia/.NET 8 desktop app for the FS25 Real FFB Bridge MVP. It provides DirectInput device scanning, Logitech wheel profile detection, safe test effects, telemetry reception, gameplay FFB calculation, and live effect status output.

Run from the repository root:

```powershell
dotnet run --project windows-app/src/App/FS25FfbBridge.App.csproj
```
